# 🎓 Student Management System

A web-based Student Management System built using Flask that allows users to efficiently manage student records.

---

## 🚀 Features

* ➕ Add new students
* ✏️ Edit student details
* ❌ Delete student records
* 🔍 Search students
* 📋 View all students in a structured table
* 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS
* **Backend:** Python (Flask)
* **Database:** SQLite
* **Server:** Gunicorn
* **Deployment:** Render
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/yashbora18/student-management-system.git
```

2. Navigate to project folder

```
cd student-management-system
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000
```
